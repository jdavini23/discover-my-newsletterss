## Getting Started

### Prerequisites
- Node.js (v18+)
- npm

### Installation
1. Clone the repository
```bash
git clone https://github.com/jdavini23/discover-my-newsletterss.git
cd discover-my-newsletterss
npm install
```

2. Start the development server
```bash
npm run dev
```

3. View Storybook
```bash
npm run storybook
```
